# cyberdr1ft3r.com

Personal cybersecurity portfolio, blog/notes, and lab website.

## Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Purpose
This project is the first version of **cyberdr1ft3r.com**: a simple static site to share cybersecurity notes, projects, and homelab/workbench experiments.

## Run locally
```bash
pnpm install
pnpm dev
```
Open http://localhost:3000

## Build
```bash
pnpm build
pnpm start
```

## Main pages
- Home
- About
- Notes
- Projects
- Contact
- Workbench
